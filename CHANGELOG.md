# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.111.0] - 2026-02-11

### Added
- Initial preparation for GitHub release.
- Project documentation (`README.md`, `CHANGELOG.md`, `.env.example`).
- Integration with Cloudflare Pages via Wrangler.
- Windows 95-style desktop environment with multiple functional apps (Instagram, Twitter, Orkut).
- Guestbook with real-time support for messages and drawings via Firebase.
- Live status integration with Last.fm and Xbox Live.

### Changed
- Migrated deployment workflow from Netlify to Cloudflare Pages.
- Optimized asset loading using Firebase Storage for static data.
- Refined UI components for better responsiveness on mobile devices.

### Fixed
- Various API error handling improvements.
- Styling consistency across different "classic" app clones.
